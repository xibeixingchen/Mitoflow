# MitoFlow验证评估指标定义

## 概述

本文档定义MitoFlow注释准确性的量化评估指标，对标PMGA论文（Plant Communications, 2025）的评估标准。

---

## 一、基因检出指标（对标PMGA Table S6）

### 1.1 基础定义

| 符号 | 含义 |
|------|------|
| TP (True Positive) | 正确检出的基因 |
| FP (False Positive) | 错误检出的基因（不存在或位置错误） |
| TN (True Negative) | 正确判断为不存在的基因 |
| FN (False Negative) | 漏检的基因 |

### 1.2 计算公式

| 指标 | 公式 | 说明 |
|------|------|------|
| **Accuracy** | (TP+TN) / (TP+FP+TN+FN) | 总体准确率 |
| **Sensitivity** | TP / (TP+FN) | 真阳性检出率（召回率） |
| **Precision** | TP / (TP+FP) | 检出结果准确率 |
| **FNR** | FN / (FN+TP) | 漏检率 |
| **F1-measure** | 2×(Precision×Sensitivity) / (Precision+Sensitivity) | 综合指标 |

### 1.3 目标值

| 指标 | PMGA表现 | MitoFlow目标 |
|------|----------|--------------|
| Accuracy | 97.4% | ≥95% |
| Sensitivity | 97.4% | ≥95% |
| Precision | 100% | ≥95% |
| FNR | 2.6% | ≤5% |
| F1-measure | 98.7% | ≥95% |

## 二、位置准确率指标

### 2.1 偏差分类

| 偏差级别 | 偏差范围 | 说明 |
|----------|----------|------|
| 精确匹配 | <50bp | 高精度定位 |
| 小偏差 | 50-100bp | 可接受的误差 |
| 中偏差 | 100-1000bp | 需改进 |
| 大偏差 | >1000bp | 严重错误 |

### 2.2 计算指标

| 指标 | 定义 | 目标值 |
|------|------|--------|
| 精确匹配率 | 基因数(<50bp) / 总基因数 | ≥75% |
| 小偏差率 | 基因数(<100bp) / 总基因数 | ≥90% |
| 平均偏差 | 所有基因位置偏差平均值 | <30bp |

### 2.3 当前表现（Round 1）

基于三方比较结果（v6）：
- 精确匹配率: 50.0%
- 小偏差率: 74.2%
- 中偏差基因: 19个
- 大偏差基因: 13个

## 三、剪接位点准确率指标

### 3.1 SSE（剪接位点错误）检测

针对多外显子基因，评估剪接位点准确性。

| 指标 | 定义 | 计算方法 |
|------|------|----------|
| SSE检出数 | 检出的正确剪接位点 | 与PMGA修正后一致 |
| SSE总数 | 基因总剪接位点数 | 从GenBank提取 |
| SSE准确率 | SSE检出数 / SSE总数 | ≥95% |

### 3.2 重点基因

PMGA识别的SSE高频基因：

| 基因 | SSE修正数 | 特点 |
|------|-----------|------|
| nad1 | 15 | 多外显子，易剪接错误 |
| nad5 | 12 | 大基因，复杂结构 |
| ccmFC | 11 | 常见剪接+终止错误 |
| nad7 | 8 | 多外显子 |
| cox2 | 5 | 典型剪接问题 |

## 四、RNA编辑识别指标

### 4.1 STGE（起始密码子获取）

RNA编辑事件C→U创造的起始密码子（ACG→M）。

| 基因 | RNA编辑起始 | NCBI表现 |
|------|-------------|----------|
| cox1 | ACG→M | 已验证 |
| nad1 | ACG→M | 已验证 |
| nad4L | ACG→M | 已验证 |
| rps10 | ACG→M | 已验证 |

**指标**: STGE检出率 = 正确识别STGE数 / 总STGE数，目标100%

### 4.2 SPGE（终止密码子获取）

RNA编辑事件C→U创造的终止密码子。

| 基因 | RNA编辑终止 | 修正数 |
|------|-------------|--------|
| atp6 | C→U创终止 | 多物种 |
| atp9 | C→U创终止 | 多物种 |
| ccmFC | C→U创终止 | 多物种 |

**指标**: SPGE检出率 = 正确识别SPGE数 / 总SPGE数，目标100%

## 五、错误类型分类（对标PMGA）

### 5.1 三类错误定义

| 错误类型 | 说明 | PMGA表现（拟南芥） |
|----------|------|-------------------|
| **A类** | 错误注释基因（漏检/误检） | AGORA:42, GeSeq:18, PMGA:1 |
| **B类** | 基因起始/终止位置错误 | AGORA:2, GeSeq:0, PMGA:2 |
| **C类** | 剪接位点位置错误 | AGORA:15, GeSeq:10, PMGA:0 |

### 5.2 MitoFlow目标

| 错误类型 | 目标错误数 | 对标PMGA |
|----------|------------|----------|
| A类 | ≤3 | PMGA:1 |
| B类 | ≤5 | PMGA:2 |
| C类 | ≤10 | PMGA:0 |

## 六、验证流程

### 6.1 单物种验证步骤

```
1. 解析三种注释来源
   - NCBI原始（GenBank）
   - PMGA修正（从corrections.csv）
   - MitoFlow输出
   
2. 基因集合比较
   - 检出基因列表
   - 计算TP/FP/FN
   
3. 位置差异分析
   - 起始/终止位置偏差
   - 分类统计
   
4. 剪接位点验证
   - 提取多外显子基因
   - 检查内含子边界
   
5. RNA编辑验证
   - 检查STGE/SPGE基因
   - 验证起始/终止密码子类型
```

### 6.2 批量验证输出

每个物种生成：
- 基因检出统计（TP/FP/FN）
- 位置偏差列表（按基因）
- 错误类型分类（A/B/C）
- SSE/STGE/SPGE检出情况

汇总输出：
- 全物种性能汇总表
- 问题基因排序表
- 改进优先级列表

## 七、达标判定标准

### 7.1 全面达标条件

所有指标同时满足：
- Accuracy ≥ 95%
- 小偏差率 ≥ 90%
- SSE准确率 ≥ 95%
- STGE检出率 = 100%
- SPGE检出率 = 100%

### 7.2 部分达标判定

| 等级 | 条件 |
|------|------|
| **优秀** | 全部达标 |
| **良好** | Accuracy ≥90%, 小偏差率 ≥85% |
| **合格** | Accuracy ≥85%, 小偏差率 ≥80% |
| **不合格** | 任一指标低于合格标准 |
